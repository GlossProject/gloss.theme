Buildout
----------
You can add `gloss.theme` to your buildout

    eggs =
        ...
        gloss.theme
 
If you're using Plone 4.3.x remember to pin version so you use the
0.7.x series NOT the 0.8.x series.

Cloud9 IDE installer
--------------------
run `installer.sh` at Cloud9::
    
    sh installer.sh
